# OpenCV Air Drawing using Hand Tracking

A real-time hand tracking and air drawing project built using Python, OpenCV, and MediaPipe.

## Features

- Real-time hand landmark detection
- Air drawing using index finger
- Webcam-based tracking
- Draw using finger movements
- Clear canvas functionality

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy

## Files

### `app.py`
Detects and displays hand landmarks using MediaPipe.

### `main.py`
Implements air drawing using index finger tracking.

## Installation

Clone the repository:

```bash
git clone https://github.com/rosa-7/Opencv-air-drawing.git
```

Install dependencies:

```bash
pip install opencv-python mediapipe numpy
```

Run the project:

```bash
python main.py
```

## Controls

- Press `q` to quit
- Press `c` to clear drawing canvas
